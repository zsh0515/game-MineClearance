# 💣 扫雷游戏 MineClearance

一个简洁美观的网页版扫雷游戏，支持最佳记录功能。

![扫雷游戏预览](https://img.shields.io/badge/version-1.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## 🎮 游戏功能

| 功能 | 说明 |
|------|------|
| **四种难度** | 初级 (9×9)、中级 (16×16)、高级 (30×16)、自定义 |
| **最佳记录** | 自动记录并显示各难度最短通关时间 |
| **智能提示** | 双击数字格自动揭开周围未标记格子 |
| **首次安全** | 第一次点击不会踩到地雷 |
| **数据持久化** | 记录保存在浏览器 localStorage |

## 🕹️ 操作方式

| 操作 | 功能 |
|------|------|
| 🖱️ 左键点击 | 揭开格子 |
| 🖱️ 右键点击 | 标记/取消标记地雷 |
| 🖱️ 双击 | 智能揭开（数字周围标记数匹配时） |
| 😊 点击表情 | 重新开始当前难度 |

## 🚀 快速开始

### 直接打开
```bash
# 直接用浏览器打开 index.html
open index.html
# 或
start index.html
```

### 使用本地服务器
```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve .

# 然后访问 http://localhost:8000
```
